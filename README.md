### requiremets

1. Python 3 <br />
2. pandas <br />
3. faker <br />
4. urllib <br />
5. numpy <br />
6. seaborn (Graphing/Plotting) <br />
7. sklearn <br />
8. pickle <br />

10. transformers (pip install -q git+https://github.com/huggingface/transformers.git) <br />
11. selenium (and the preferred web driver)

go to 'Malicious URL Detection' folder  <br /> <br />
